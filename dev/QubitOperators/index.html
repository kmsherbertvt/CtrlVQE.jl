CtrlVQE.QubitOperators — Module
QubitOperators

Interfaces arbitrary physical Hilbert space with a strictly binary logical Hilbert space.

NOTE: I don't especially like how this module is organized, so consider this code subject to change.

CtrlVQE.QubitOperators.nqubits — Method
nqubits(H)

Infer the number of qubits of a statevector or matrix living in a logical qubit space.

In other words, calculates n assuming the dimension of H is 2^n.

CtrlVQE.QubitOperators.qubitisometry — Method
qubitisometry(device::Devices.DeviceType)

An isometry from the physical Hilbert space of a device onto a logical two-level space.

If Φ is the isometry, and |ψ⟩ is a statevector living in the full Hilbert space, the vector Φ|ψ⟩ is a smaller vector living in the two-level space.

source
CtrlVQE.QubitOperators.qubitprojector — Method
qubitprojector(device::Devices.DeviceType)

A projector from the physical Hilbert space of a device onto a logical two-level space.

The projector Π does not reduce the size of its operand; it only removes support on the non-logical states. If you want to change the size, use qubitisometry instead.
